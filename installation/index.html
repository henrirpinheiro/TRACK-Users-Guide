    Installation procedure¶

    To configure and install the TRACK code, the following software are needed:

    Required software¶

    1. gcc compiler
    2. gfortran compiler
    3. NetCDF (Network Common Data Form) software libraries

    Optional but strongly recommended¶

    1. CDO – Climate Data Operators for managing NetCDF files
    2. NCO – NetCDF Operators for managing NetCDF files
    3. GrADS or another visualization tool

    Step 1 – Go to the TRACK directory¶

    Navigate to the installation folder:

    cd TRACK-1.5.2

    Ensure the current directory . is in your PATH environment variable:

    export PATH=${PATH}:.

    Step 2 – Set environment variables (bash)¶

    In bash, export the following variables:

    export CC=gcc
export FC=gfortran
export ARFLAGS=
export NETCDF=[path_to_your_netcdf_installation]

    Replace [path_to_your_netcdf_installation] with the actual location of your NetCDF installation.

    Step 3 – Build the main executable¶

    Use the following command to build the executable (you can change linux to another name if desired):

    master -build -i=linux -f=linux

    Step 4 – Compile utility functions¶

    Run the following to compile the utility programs:

    make utils

    Step 5 – Optional: Install in a custom directory¶

    If you want to install the code outside of your $HOME directory, use:

    master -build -i=linux -f=linux -u=[path]

    Replace [path] with the directory path where the TRACK directory is located.

    Final step – verify installation¶

    If everything is correctly configured, the executable track.linux will be created in the bin/ directory.

    ✅ You are now ready to run the TRACK code!
