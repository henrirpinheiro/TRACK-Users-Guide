    Data Pre-processing

    Data Pre-processing

    After generating the vorticity data, this step prepares it for spectral filtering and spectral truncation.

    The data are initially truncated to a triangular truncation on a Gaussian grid, focusing on synoptic-scale features.
    - Vorticity fields: typically truncated at T42, as this is a noisy variable.
    - MSLP or geopotential height fields: may use T63 or T106.

    The large-scale background is removed using spherical harmonic decomposition, setting coefficients of wavenumbers ‚â§ 5 to zero.

    üîß Running Cholesky Pre-processing¶

    This operation uses the configuration file cholesky_ERA5.in, located in the main directory.
    It needs to be run only once, although the process may take some time.

    bin/track.linux -i ERA5_VOR850_MAM1979.dat -f chol < cholesky_ERA5.in

    Alternatively, to answer the prompts manually, run:

    bin/track.linux -i ERA5_VOR850_MAM1979.dat -f chol

    ‚ûï Download cholesky_ERA5.in

    Grid Size Example

    1440 and 719
    These values in the cholesky_ERA5.in file correspond to the X and Y grid dimensions, excluding the poles
    (i.e., 721 - 2 = 719 in the Y direction).

    üì§ Output Files from Cholesky¶

    • A binary file named specfil.chol_band000 will be created in the outdat/ directory.
    • Another file will be saved in the spectral/ directory, named based on the last line in the input file.
      E.g., ERA5-T42-1440-719.

    You may use a custom filename for the input file, but the command syntax must remain the same.

    üåê Running Spectral Filtering¶

    Next, perform spectral filtering using the following command:

    bin/track.linux -i ERA5_VOR850_MAM1979.dat -f filt < specfilt_ERA5.in

    ‚ûï Download specfilt_ERA5.in

    Don't worry about warnings

    You may see error messages during execution ‚Äî these are expected and not a problem.

    üì§ Output File¶

    After filtering, a binary file named:

    specfil.filt_band000

    ...will be created in the outdat/ directory.

    Move this file to the indat/ directory and rename it (e.g., ERA5_VOR850_MAM1979_filt.dat).
    This will be your vorticity data truncated at T42 resolution, ready for feature tracking.
