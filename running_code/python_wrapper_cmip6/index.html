    Python Wrapper for CMIP6¶

    For users interested in applying TRACK to CMIP6 simulations, the pyTRACK-CMIP6 repository provides a set of Python scripts and tools that simplify the process of running TRACK on large climate model datasets — especially for extratropical cyclone tracking.

    Key Features¶

    • Wrapper scripts to automate TRACK configuration for multiple CMIP6 models or ensemble members
    • Batch processing capabilities across historical and future experiments (e.g., SSP585)
    • Pre-processing utilities for vorticity fields and regridding

    Repository¶

    👉 pyTRACK-CMIP6 on GitHub

    When to Use This Wrapper¶

    Use this wrapper if you are:

    • Working specifically with CMIP6 climate model outputs
    • Analyzing multi-model ensembles or large-scale simulations
    • Interested in reproducible pipelines to generate TRACK-compatible input and process the output efficiently

    Documentation and Examples¶

    Detailed instructions, including example configurations, input formats, and expected outputs, are provided in the repository documentation.

    Note: A functional TRACK installation is required to use this wrapper.
